## **Card Specification Document**

Every card needs:

* Name
* Type (Combat / Negotiation / Thread)
* Clan/Class associations
* Cost
* Effect text
* Tags (Fire, Frost, Shadow, Divine, etc.)
* Synergies
* Unlock method
* Rarity tier
* Upgrade track
* Visual reference (placeholder OK)
